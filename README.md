# SutraFi
A privacy-first financial planning platform designed for Indian users, combining AI-driven insights with interactive financial tools to make money management more structured, accessible, and engaging.
SutraFi helps users evaluate their financial health, optimize taxes, analyze investments, and plan long-term wealth goals through a unified platform built entirely on the client side.
## Overview
Managing personal finances often requires switching between multiple tools for budgeting, tax planning, investment tracking, and retirement calculation. SutraFi brings these functions into a single platform, allowing users to:
 * **Understand** their current financial position.
 * **Identify** areas for improvement.
 * **Optimize** tax planning decisions.
 * **Analyze** portfolio allocation and risk.
 * **Plan** for long-term financial independence.
The platform is designed specifically for Indian financial workflows and regulations.
## Core Features
### 📊 Financial Health Score
SutraFi provides a consolidated financial score based on six essential dimensions:
 * Income Stability
 * Savings
 * Spending Efficiency
 * Investment Preparedness
 * Debt Management
 * Risk Protection
> This score offers users a quick summary of their financial condition along with category-level insights.
> 
### 💰 Tax Planning Assistant
A dedicated tax module tailored for Indian taxpayers. Capabilities include:
 * Old vs. New Tax Regime comparison
 * Estimated tax calculation
 * Deduction identification
 * Tax-saving recommendations
 * Investment-linked tax planning support
### 📈 Portfolio Analysis
Users can evaluate their investment portfolio through allocation and diversification analysis. Key insights include:
 * Asset distribution breakdown
 * Diversification review
 * Sector concentration analysis
 * Risk exposure summary
 * Improvement suggestions
### FIRE Planner
A planning tool for users targeting Financial Independence and Early Retirement. Includes:
 * Retirement corpus estimation
 * Savings target calculations
 * Inflation-adjusted projections
 * Goal timeline simulation
### 🎮 Interactive User Experience
To improve engagement and retention, SutraFi introduces a mission-based user journey inspired by strategic progression systems. Users can track:
 * Financial milestones
 * Goal completion progress
 * Achievement-based progress indicators
## Privacy and Data Security
SutraFi is built with a strict privacy-first architecture:
 * **No backend server dependency**
 * **No cloud data storage**
 * **No financial data transmission**
 * **100% client-side processing**
 * **Data stored locally on the user’s device**
This ensures that all sensitive financial information remains fully under user control.

## Installation
Get the project up and running locally on your machine:
```bash
# Clone the repository
git clone https://github.com/yourusername/sutrafi.git
# Navigate to the project directory
cd sutrafi
# Install dependencies
npm install
# Run the development server
npm run dev
```
